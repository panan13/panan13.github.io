Verilog 中的运算符

算术运算符

  • + : 加法
  • - : 减法
  • * : 乘法
  • / : 除法
  • % : 取模

比较运算符

  • == : 相等
  • != : 不等
  • < : 小于
  • <= : 小于等于
  • > : 大于
  • >= : 大于等于

逻辑运算符

  • && : 逻辑与
  • || : 逻辑或
  • ! : 逻辑非

按位运算符

  • & : 按位与
  • | : 按位或
  • ^ : 按位异或
  • ~ : 按位非
  • << : 左移
  • >> : 右移

连接运算符

  • {} : 连接操作符，例如 {2'b10, 2'b11} 会连接成 4'b1011

条件运算符

  • ?: : 条件操作符，例如 a ? b : c 表示如果 a 为真，则选择 b，否则选择 c
